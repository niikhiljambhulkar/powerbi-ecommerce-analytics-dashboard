# E-Commerce Analytics Dashboard

## Project Overview

This project is an interactive Power BI dashboard built to analyze e-commerce sales, profitability, customer behavior, and product performance.

## Tools Used

* Power BI
* Power Query
* DAX

## Features

* KPI Cards
* Interactive Slicers
* Multi-page Dashboard
* Drill-through Analysis
* Tooltips
* Mobile Layout

## Business Insights

* West region generated highest sales
* Technology category contributed maximum revenue
* Some sub-categories showed low profitability
* Standard Class was the most used shipping mode

## Dashboard Pages

1. Overview
2. Product Analysis
3. Customer Analysis
4. Profit Analysis
